# make something

Build your first app with AI.

## what is this?

This is a project created in a single session of makesomething weekly AI-driven development.
The goal of these sessions is to build an idea publicly and publish it.

## idea
A web-app that lets you define your reel idea and upload the audio voiceover for the reel. It can then auto-generate the reel video by merging anime clips from the internet and adds captions to it as well.
